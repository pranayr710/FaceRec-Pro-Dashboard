# Emotion Detector Intelligence

A custom-built deep learning model for facial expression recognition. This module integrates directly with the main dashboard to provide real-time emotional state analytics.

## Core Capabilities
- **Facial Expression Mapping**: Analyzes landmarks to determine valence and arousal.
- **Temporal Smoothing**: Ensures stable emotion labels across video frames.
- **Deep Integration**: Feed-forward logic to the centralized psychometric scoring engine.
