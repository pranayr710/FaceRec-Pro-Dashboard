# Eye Tracking and Pose Analytics

Deep implementation of pupil detection and 3D head pose estimation using the Perspective-n-Point (PnP) method. This module provides the core geometric intelligence for attention tracking and engagement scoring.

## Features
- Real-time pupil centroid detection.
- 3D Head Pose calculation (Yaw, Pitch, Roll).
- Landmark-based geometric modeling.

## Technical Details
The pose computation utilizes standard camera calibration and dlib facial landmarks to project 2D image points into 3D world coordinates.
